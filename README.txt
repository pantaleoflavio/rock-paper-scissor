### Rock Paper Scissor (Lizard Spock)

a nice variant of the classic rock paper scissors game, inspired by the TV series Big Bang Theory

HTML/CSS part where the style of the page is built, then Javascript: for each player choice, there is a DOM modification and CPU random choice.
with a some functions and logic, it is checked the style status of the selected icon, there is a random answer of CPU (with same logic for style change of icons),
DOM modification for the score update and reset of the game.
